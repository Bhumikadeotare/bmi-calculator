# BMI Calculator

## Project Overview

This project is a Body Mass Index (BMI) Calculator developed using Python. The application allows users to enter their weight and height, calculates their BMI using the standard formula, and displays the corresponding health category.

---

## Features

- Calculates BMI accurately
- User-friendly interface
- Input validation
- Displays BMI category
- Handles invalid inputs gracefully

---

## Technologies Used

- Python 3
- Tkinter (for GUI version)

---

## BMI Formula

BMI = Weight (kg) / Height² (m²)

---

## BMI Categories

| BMI Range | Category |
|-----------|----------|
| Less than 18.5 | Underweight |
| 18.5 – 24.9 | Normal Weight |
| 25.0 – 29.9 | Overweight |
| 30.0 and above | Obese |

## How to Run the Project

1. Install Python 3.
2. Download or clone this repository.
3. Open the project folder in VS Code.
4. Run the following command:

```bash
python bmi_calculator.py
```

5. Enter your weight and height.
6. View your BMI and health category.
